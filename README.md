# Youtube動画でシャドーイング勉強

Youtube動画でシャドーイングやリピーティングしたいなとおもってVibe codingさせたChrome拡張です。

## どんな機能があるか
- youtubeをフレーズごとに分割
- リピーティング機能
  - フレーズごとに2回再生される
  - 録音モードになるので、そのフレーズを自分の声で発してみる
  - 録音した声が流れるので聞く
  - 次のフレーズに進む

# その他
- Youtubeの動画をダウンロードすることなく使えます
- 自分の声もインメモリ保存なので、ブラウザ閉じたら消えます。
今は安心安全で使えるのがいいかと思った
